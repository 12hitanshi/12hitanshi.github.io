Project 3: Python

This Python code generates a color palette for a brand based on its core values.

Input:

  • Brand name
  • Primary aim/value (e.g., sustainability, innovation, luxury)

Output:

A tailored color palette; if the aim is sustainability, an additional contrasting color is suggested.

Example : Input: Name of my brand - etheria Aim of my Brand - Sustainability Output: Color Palette - Green, Blue, Beige
